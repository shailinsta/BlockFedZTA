# BlockFedZTA Dataset and Results

## Description
This repository contains dataset and experimental results for:
Trust-Based Federated Learning for Intrusion Detection Systems.

## Contents
- Final_5Class_IDS.csv : Balanced dataset
- summary.csv : Main results
- folds.csv : Cross-validation results
- trust.csv : Client trust scores
- scalability.csv : Scalability analysis
- predictions.csv : Model predictions
- confusion_matrix.csv : Confusion matrix

## Method
- Federated Learning with Trust-Based Aggregation
- Compared against FedAvg baseline
